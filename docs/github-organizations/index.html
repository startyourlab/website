GitHub Organizations

To work on projects as a team effectively and efficiently, your lab needs a GitHub organization account. An organization serves as your lab's shared account to facilitate collaboration on multiple projects at once. For example, any new collaborators or lab members can simply be added to the organization with one or two clicks, and thereby be given access to all GitHub-related materials for the lab. Once you create your lab's organization, you can then invite your lab members who have already created GitHub user accounts to join.
